CGL 도입 배경 Why CGL? — 기존 배상책임보험의 한계와 CGL 전환 근거
기존 국문 배상책임보험 — 한계
담보 구조 문제
  • 열거형 담보 — 명시된 사고 유형만 보상
  • 보상 공백 발생 (명시되지 않은 사고 → 자동 거절)
  • 면책 판단 지연 → 입주민 불만 누적
운영 효율 문제
  • 수기 처리 — 접수·산정·결재 모두 수작업
  • 법적 근거 부재로 의견서 발송 불가
  • 구상권 행사 체계 미흡
CGL (Commercial General Liability) — 구조 전환
포괄형 담보 원칙: "모든 사고를 보상하되, 면책 사유만 열거한다" — 보상 공백 원천 차단
PILLAR A
신체장해 (BI)
아파트 내 다침·부상 발생 시 인적 손해 보상. 공용부·전유부 모두 포함.
PILLAR B
인격권침해
소송방어비용 자동 포함. 면책의견서 즉시 발송 법적 근거 제공.
PILLAR C
구내치료비
과실 여부 무관, 1인당 최대 100만원 즉시 지급. 청구 절차 최소화.
CGL → TYPE A/B/C 자동 분류 근거
분류 CGL 근거 주요 내용 처리 방향
TYPE A CGL 미적용 하자보증기간 내 시공사 귀책 시공사 하자보수 요청
TYPE B Pillar B — 인격권침해 AI 면책 판단 + 의견서 자동 발송 면책 통보 / 이의신청 처리
TYPE C Pillar A / C — 신체·구내치료 보험 적용, 보험금 산출 및 지급 적산 → 승인 → 지급
핵심 효과: CGL 도입으로 열거형 → 포괄형 전환, AI 자동 분류 법적 근거 확보, 면책의견서 즉시 발송 가능, 구상권 행사 체계화
